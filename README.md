# Angetic Essence PWA

Mobile Progressive Web App (PWA) version of [Angetic Essence](https://laikaseriousjames-barkdog.github.io/Angetic-Essence/).

## Features
- Installable on Android (Add to Home Screen)
- Offline support via Service Worker
- Mobile-optimized dark UI
- Full bottom navigation

## Deploy
Hosted on Netlify. One-click deploy or drag-and-drop the folder at [netlify.com](https://netlify.com).
